Privacy in Machine Learning

Reading list (MIAs, FL, Gradient Inversion, DP)

A curated set of papers that match the lecture flow: membership inference → federated learning → gradient inversion → differential privacy / DP-SGD.

Membership Inference Attacks against Machine Learning Models

Reza Shokri, Marco Stronati, Congzheng Song, Vitaly Shmatikov • IEEE S&P 2017 MIA Black-box
PDF arXiv
The classic paper: shadow models + attack classifier, showing membership leakage from confidence vectors; discusses mitigations.
BibTeX 
@inproceedings{shokri2017mia,
  title={Membership Inference Attacks against Machine Learning Models},
  author={Shokri, Reza and Stronati, Marco and Song, Congzheng and Shmatikov, Vitaly},
  booktitle={IEEE Symposium on Security and Privacy (S\&P)},
  year={2017}
}

Deep Leakage from Gradients

Ligeng Zhu, Zhijian Liu, Song Han • NeurIPS 2019 Gradient inversion Reconstruction
PDF arXiv
Shows how shared gradients can be matched by optimizing dummy inputs/labels, recovering training examples (images/text).
BibTeX 
@inproceedings{zhu2019dlg,
  title={Deep Leakage from Gradients},
  author={Zhu, Ligeng and Liu, Zhijian and Han, Song},
  booktitle={Advances in Neural Information Processing Systems (NeurIPS)},
  year={2019}
}

Inverting Gradients — How easy is it to break privacy in federated learning?

Jonas Geiping, Hartmut Bauermeister, Hannah Dröge, Michael Moeller • NeurIPS 2020 Federated Learning Gradient inversion
PDF arXiv Code
Stronger, more practical gradient inversion; analyzes when reconstruction is easy/hard in realistic FL settings.
BibTeX 
@inproceedings{geiping2020inverting,
  title={Inverting Gradients -- How easy is it to break privacy in federated learning?},
  author={Geiping, Jonas and Bauermeister, Hartmut and Dr{\"o}ge, Hannah and Moeller, Michael},
  booktitle={Advances in Neural Information Processing Systems (NeurIPS)},
  year={2020}
}

Deep Learning with Differential Privacy

Martín Abadi et al. • CCS 2016 Differential Privacy DP-SGD
PDF arXiv ACM DOI
Introduces DP-SGD and the Moments Accountant analysis; the “classic” DP deep learning reference.
BibTeX 
@inproceedings{abadi2016dpdl,
  title={Deep Learning with Differential Privacy},
  author={Abadi, Martín and Chu, Andy and Goodfellow, Ian and McMahan, H. Brendan and Mironov, Ilya and Talwar, Kunal and Zhang, Li},
  booktitle={ACM Conference on Computer and Communications Security (CCS)},
  year={2016}
}

Scalable Private Learning with PATE

Nicolas Papernot, Shuang Song, Ilya Mironov, Ananth Raghunathan, Kunal Talwar, Úlfar Erlingsson • 2018 Differential Privacy Knowledge transfer
PDF arXiv
DP via noisy teacher voting + student training (knowledge distillation). A very different privacy mechanism than DP-SGD.
BibTeX 
@article{papernot2018pate,
  title={Scalable Private Learning with {PATE}},
  author={Papernot, Nicolas and Song, Shuang and Mironov, Ilya and Raghunathan, Ananth and Talwar, Kunal and Erlingsson, {\'U}lfar},
  journal={arXiv preprint arXiv:1802.08908},
  year={2018}
}

SoK: Membership Inference is Harder Than Previously Thought

Antreas Dionysiou, Elias Athanasopoulos • PETS 2023 MIA SoK / evaluation
PDF
A useful “reality check” paper: when MIAs succeed/fail, what assumptions they rely on, and evaluation pitfalls.
